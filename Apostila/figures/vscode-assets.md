# Figuras do Visual Studio Code

Arquivos baixados ou gerados para a seção de instalação do Visual Studio Code.

- `vscode-logo.png` e `vscode-logo.svg`: ícones oficiais baixados da página de diretrizes de marca do Visual Studio Code.
- `vscode-download-page.png`: captura gerada localmente, com Chrome em modo headless, da página oficial de download.
- `vscode-user-interface.png`: imagem oficial da documentação do Visual Studio Code sobre a interface do usuário.
- `vscode-extensions-icon.png`: ícone oficial da documentação do Visual Studio Code para a área de extensões.
- `vhdl-for-professionals-marketplace.png`: captura gerada localmente, com Chrome em modo headless, da página oficial da extensão no Marketplace.
- `vhdl-for-professionals-marketplace-crop.png`: recorte da captura do Marketplace, usado no capítulo para destacar nome, publicador e botão de instalação.
- `vhdl-for-professionals-highlight.png`: imagem pública do repositório da extensão, usada como exemplo visual de apoio à edição de código VHDL.
- `vscode-opengraph-home.png`: imagem oficial usada pela página inicial do Visual Studio Code para pré-visualização. Disponível como referência, mas não inserida no capítulo por ter caráter mais promocional.

Fontes:

- https://code.visualstudio.com/brand
- https://code.visualstudio.com/download
- https://code.visualstudio.com/docs/getstarted/userinterface
- https://code.visualstudio.com/docs/configure/extensions/extension-marketplace
- https://marketplace.visualstudio.com/items?itemName=ViDE-Software.v4pvhdlforprofessionals
- https://bitbucket.org/ferringer/v4p_public/raw/master/highlight-error-semantic.png

Observação: a página de marca do VS Code permite o uso do ícone em documentação e tutoriais sobre o Visual Studio Code, sem modificar ou redesenhar o ícone.
