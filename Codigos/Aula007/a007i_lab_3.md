# Exercício: Identificador de número da sequencia de Fibonacci

**Objetivo:** Crie um código em VHDL que indique se as chaves do kit de10 lite representam um número da sequência de Fibonacci. Caso o número seja da sequência, acenda um LED específico. A sequência de Fibonacci é definida como:
- F(0) = 0
- F(1) = 1
- F(n) = F(n-1) + F(n-2) para n > 1
