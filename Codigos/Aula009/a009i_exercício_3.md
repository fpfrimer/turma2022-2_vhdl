# Exercício: Gerador de Sinal PWM

**Objetivo:** Criar um procedimento que gere um sinal PWM (modulação por largura de pulso) com base nos valores de entrada de período e duty cycle. O procedimento deve aceitar o período total em ciclos de clock e o duty cycle como porcentagem (valor de 0 a 100) e gerar o sinal PWM correspondente.


**Desafio**: Incluir uma funcionalidade para ajustar a frequência do sinal PWM dinamicamente.
